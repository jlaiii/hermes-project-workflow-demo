# hermes-project-workflow-demo

**Live project page:** https://jlaiii.github.io/hermes-project-workflow-demo/

This repo demonstrates the standard Hermes project workflow:

1. 📝 **README created** — with description, links, and setup notes
2. 🌐 **index.html project page** — auto-generated landing page
3. ☁️ **GitHub Pages enabled** — public URL on every push

## What This Workflow Does

For every project we build together, Hermes will:
- Create a new GitHub repo
- Write a descriptive README
- Generate an `index.html` project page (even for non-web projects)
- Enable GitHub Pages so you get a live URL
- Link the Pages URL in the repo description

The `index.html` acts as a **project dashboard** — showing what the project does, screenshots, links, and status.
